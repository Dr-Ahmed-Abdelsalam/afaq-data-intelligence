# Afaq Data Intelligence  
## آفاق لحلول البيانات والذكاء الاصطناعي

**Afaq Data Intelligence** is a professional landing page and early-stage digital platform for data intelligence, business analytics, AI automation, and digital transformation services.

The project is designed to help organizations transform scattered data into actionable insights, dashboards, reports, and AI-powered decision-support tools.

## 📩 Functional Forms

The website includes working forms connected to Supabase:

### 1. Demo Request Form

Stores client demo requests in the `leads` table and sends an automatic email notification.

### 2. Expert Registration Form

Stores expert applications in the `experts` table and sends an automatic email notification.

### 3. Service Request Form

Stores service requests in the `service_requests` table and sends an automatic email notification.

---

## 🔐 Security Notes

This repository does not include private API keys or service-role secrets.

Public frontend configuration may include a Supabase publishable key, while sensitive keys such as email API keys and webhook secrets are stored securely in Supabase Edge Function Secrets.
